# NOIRVOID — GitHub Pages

Arquivos separados para publicar no GitHub Pages.

## Estrutura

```txt
index.html
css/style.css
js/tailwind-config.js
js/script.js
assets/infinite-void-frente.png
assets/infinite-void-verso.png
```

## Publicar

1. Suba todos os arquivos e pastas na raiz do repositório.
2. No GitHub, acesse Settings > Pages.
3. Selecione Deploy from a branch > main > /root.

## Infinite Void Tee

Na vitrine, o card mostra o verso por padrão e troca para a frente no hover.
